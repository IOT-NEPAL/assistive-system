Accessibility Features Summary

  • Disability Selection Modal: Choose your specific accessibility needs
  • Colorblind Simulation: Test how the site looks with different types of colorblindness
  • High Contrast Mode: Enhanced contrast for better visibility
  • Font Size Control: Adjustable text sizes
  • Screen Reader Support: Enhanced ARIA labels and announcements
  • Keyboard Navigation: Full keyboard accessibility
  • Touch Target Enhancement: Larger touch targets for motor impairments
  • Visual Feedback: Enhanced visual indicators for hearing impairments
  • Simplified Interface: Reduced complexity for cognitive differences
